# Phase 3 — Hyper-Automation at Scale (Months 20–36)

Full narrative: see [README.md](../README.md#phase-3--hyper-automation-at-scale-months-20-36).

## Implementation Checklist
- [ ] Run process mining across candidate end-to-end processes
- [ ] Prioritize 3–5 flagship processes (order-to-cash, procure-to-pay, hire-to-retire, incident-to-resolution)
- [ ] Redesign/simplify each process before automating (no "paving the cow path")
- [ ] Deploy intelligent document processing (IDP) for unstructured inputs
- [ ] Unify RPA bots, APIs, and AI agents under one orchestration plane
- [ ] Launch internal AI copilots for employee-facing workflows
- [ ] Build automation observability dashboard (bot health, exceptions, SLAs)
- [ ] Track automation ROI centrally, not per-team
