# Phase 1 — Data & Governance Foundation (Months 1–12)

Full narrative: see [README.md](../README.md#phase-1--foundation-data--governance-months-1-12).

## Implementation Checklist
- [ ] Complete data inventory across ERP, CRM, SCM, HRIS, and shadow-IT sources
- [ ] Stand up cloud data lakehouse and CDC ingestion pipelines
- [ ] Deploy MDM for customer, product, vendor domains
- [ ] Launch data governance council with named stewards per domain
- [ ] Classify all datasets by sensitivity (PII/PCI/confidential/public)
- [ ] Ratify Responsible AI charter v1 with Legal/Risk/Compliance
- [ ] Staff AI Center of Excellence (ML eng, data eng, MLOps, risk liaison)
- [ ] Ship 3–5 quick-win RPA/AI pilots to build momentum and self-fund Phase 2

## Sample 90-Day Kickoff Plan
| Days | Focus |
|---|---|
| 1–30 | Data inventory, governance council charter, CoE hiring plan |
| 31–60 | Lakehouse architecture design, MDM vendor selection, first quick-win pilot scoped |
| 61–90 | Ingestion pipelines live for top 3 domains, governance council first ruling, pilot in build |
